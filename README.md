# Spotify-clon

Un clon de Spotify construido con React y Vite. Proyecto para practicar React, componentes reutilizables y gestión de estado.

## Stack Tecnológico

- **React** - Librería de UI
- **Vite** - Build tool y dev server
- **ESLint** - Análisis de código

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Vista previa

```bash
npm run preview
```
